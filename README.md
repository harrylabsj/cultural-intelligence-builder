# Cultural Intelligence Builder
培养跨文化意识、跨文化沟通技能和全球思维适应力，帮你在多元文化环境中从容应对。

## What it does
- 🌍 文化编程审计：识别你自身被植入的文化假设（时间观、等级观、沟通方式）
- 📖 CQ 知识框架：介绍高语境/低语境沟通、权力距离、个人主义等实用文化维度
- 🤝 跨文化沟通实战：提供会前准备模板、实时沟通调整策略和关系修复方案

## Example scenarios

**1. 中国人在外企怎么和外国同事相处**
> 👤 我在外企工作，同事们是美国人/欧洲人/印度人，每次开会我都不知道他们到底想表达什么，文化差异好大。怎么办？
> 🤖 外企跨文化沟通指南——美国人：直接说观点，反对用"I have a different thought"开头；欧洲人：会议前先发邮件，讨论注意逻辑链条；印度人：点头≠同意，需明确确认"So we agree that..."。通用技巧：不懂直接问"Could you elaborate on that?"。关键：文化差异不是对错，是习惯不同。

**2. 管理跨国团队的领导力**
> 👤 I'm leading a team across US, India, Japan, and Germany. Build my cultural intelligence.
> 🤖 CQ发展计划——文化维度映射（Hofstede各国权力距离、个人主义、不确定性规避），沟通风格适应指南，各国会议引导调整建议，每周跨文化反思日志模板。

**3. 公司全球化新人培训方案**
> 👤 Create a cultural intelligence onboarding program for our company's global new hires.
> 🤖 90天CQ入职课程——第1月：自我觉察（个人文化视角、隐含偏见审计）；第2月：知识（各国文化深度解析、区域商务礼仪）；第3月：策略+行动（跨文化情景模拟、文化导师配对、真实项目应用）。
